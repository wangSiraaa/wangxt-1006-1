## 1. 产品概述

早教试听预约系统，按顾问接待家长的实际节奏设计，覆盖从课程维护、家长预约、候补排队到试听反馈、转正跟进的全流程。解决早教机构试听课预约管理混乱、年龄不匹配、爽约频发、候补无序等痛点，目标用户为早教机构的顾问、家长和授课老师。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 顾问 | 选择顾问身份进入 | 维护体验课、月龄段、能力标签、老师排班、教室容量；管理候补队列、黑名单、优惠券、转正跟进 |
| 家长 | 选择家长身份进入 | 选择宝宝年龄和试听时段预约；查看预约状态、候补位置、爽约冻结提醒 |
| 老师 | 选择老师身份进入 | 查看试听名单、填写体验报告、标记出勤状态 |

### 2.2 功能模块

1. **顾问维护端**：体验课管理、月龄段配置、课程能力标签、老师排班、教室容量设置
2. **家长预约端**：宝宝信息录入、年龄选择、时段浏览、预约提交、候补排队
3. **老师名单端**：试听名册查看、出勤签到、体验报告填写
4. **候补队列**：满员候补、候补排队、候补自动转正
5. **试听反馈**：体验报告录入、宝宝表现评分、课程匹配建议
6. **转正漏斗**：试听→意向→签约全流程追踪、转化率统计
7. **频次限制**：手机号每周一次限制、爽约冻结、黑名单管理
8. **操作审计**：关键操作日志记录、预约/取消/爽约/转正时间线

## 3. 核心流程

### 预约流程
家长进入预约页 → 输入宝宝信息（姓名、月龄、手机号）→ 选择试听课程和时段 → 系统校验（年龄匹配、手机号频次、是否黑名单/冻结）→ 有名额则预约成功，名额满则进入候补 → 预约成功后家长和老师均可查看

### 候补转正流程
名额满 → 家长进入候补队列 → 有人取消或爽约释放名额 → 候补队列首位自动转正 → 通知家长

### 爽约冻结流程
老师标记爽约 → 系统自动冻结该手机号指定天数 → 冻结期间无法预约 → 冻结期满自动解冻

### 转正跟进流程
试听完成 → 顾问查看试听反馈 → 发放优惠券 → 沟通意向 → 标记签约/流失

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖橙 (#F97316) + 柔米白 (#FFFBF5)，传达温暖、亲和的早教氛围
- **辅助色**：薄荷绿 (#10B981) 用于成功状态，珊瑚红 (#EF4444) 用于警告/错误，天蓝 (#3B82F6) 用于信息提示
- **按钮风格**：圆角 (rounded-xl)，柔和阴影，hover 时轻微上浮
- **字体**：标题用 Noto Sans SC Bold，正文用 Noto Sans SC Regular
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式内容展示
- **图标风格**：Lucide 图标，线条风格，与整体清新感统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 顾问工作台 | 体验课管理 | 卡片网格展示课程，编辑弹窗，月龄段多选标签 |
| 顾问工作台 | 排班日历 | 周视图日历格子，拖拽排班，教室容量数字角标 |
| 家长预约页 | 宝宝信息 | 表单输入，月龄滑块选择器，手机号输入 |
| 家长预约页 | 时段选择 | 日期横向滚动 + 时段卡片网格，名额进度条 |
| 家长预约页 | 状态提示 | Toast 提示年龄不匹配，冻结状态横幅，候补位置指示器 |
| 老师名单页 | 试听名册 | 表格列表，出勤状态切换按钮，体验报告弹窗 |
| 候补管理页 | 候补队列 | 各时段候补卡片列表，拖拽排序，转正按钮 |

| 操作审计页 | 审计日志 | 时间线视图，操作类型筛选标签 |

### 4.3 响应式设计

- 桌面端优先设计（1280px+）
- 平板端适配（768px-1279px）：左侧导航收起为图标模式
- 移动端适配（<768px）：底部Tab导航，卡片单列布局

### 4.4 运行样例场景

系统预置以下演示数据，启动后可直接体验：

1. **年龄不匹配**：选择18月龄宝宝预约"6-12月专属课"，弹出警告但允许继续
2. **满员候补**：某时段名额已满（容量3人已预约3人），新预约自动进入候补
3. **手机号频次限制**：同一手机号本周已预约过，再次预约时被拦截
4. **爽约冻结**：老师标记某宝宝爽约后，该手机号7天内无法预约，页面显示冻结横幅
5. **候补自动转正**：爽约释放名额后，候补首位自动转正为正式预约
